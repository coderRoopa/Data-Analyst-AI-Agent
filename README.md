# Data-Analyst-AI-Agent
NLP-powered analytics tool that converts natural language queries into SQL using Google Gemini and executes them on datasets via DuckDB, delivering results with automated insights. This project enables non-technical users to analyze data using plain English
An AI-powered web application that allows users to analyze CSV or Excel datasets by asking natural language questions. The app automatically generates SQL queries using Google Gemini, executes them on DuckDB, and provides human-readable explanations.
